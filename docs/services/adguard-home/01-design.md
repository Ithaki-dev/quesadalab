## Objetivo

Proporcionar un servidor DNS centralizado para toda la red doméstica.

El servicio permite:

- Bloqueo de publicidad.
- Bloqueo de rastreadores.
- Protección contra dominios maliciosos.
- Estadísticas DNS.
- Resolución local.

---

## Razón de la elección

Se seleccionó AdGuard Home por las siguientes razones:

- Código abierto.
- Bajo consumo de recursos.
- Excelente interfaz web.
- Fácil integración con OpenWrt.
- Soporte para DNS-over-HTTPS.
- Soporte para DNS-over-TLS.
