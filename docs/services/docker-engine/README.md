# Docker Engine

Docker Engine is the container platform used by QuesadaLab.

## Overview

Docker runs inside a dedicated Debian 13 virtual machine hosted on Proxmox VE.

## Host Information

| Property | Value |
|----------|-------|
| Hostname | docker01 |
| Operating System | Debian 13 |
| IP Address | 192.168.1.30 |
| Virtualization | KVM |
| Storage | 80 GB |
| Runtime | Docker Engine |
